Finding 1. Rectal NET is not colorectal cancer

Hypothesis (from Anveshar): rectal neuroendocrine tumors are a different disease from the colorectal adenocarcinoma of the same organ. Test it on real data.

notebook 07, the actual code
# driver mutation frequency in NET vs adenocarcinoma
def mut_freq(study, ids):
    body = {"sampleIds": ids, "entrezGeneIds": list(PANEL)}
    m = post(f"{API}/molecular-profiles/{study}_mutations/mutations/fetch", body)
    s = defaultdict(set)
    for x in m: s[PANEL[x["entrezGeneId"]]].add(x["sampleId"])
    return {g: 100*len(s[g])/len(ids) for g in PANEL.values()}

result: driver frequency (%), colored by value

Transcriptomics: only 3.7% of colorectal adenocarcinomas are neuroendocrine-high.

The driver landscape tracks histology, not organ. A rectal NET matches a pancreatic NET, not the colorectal cancer next to it, so it needs neuroendocrine therapy (SSTR2, Lu-177 DOTATATE), not the colorectal toolkit.

Finding 2. Integrated multi-omics beats a mutation call (uveal melanoma)

Hypothesis: an integrated multi-omic class (DNA + RNA + protein) predicts uveal melanoma survival better than a single BAP1 mutation call. Test it on TCGA-UVM.

notebook 08, the actual code
# integrated class from a prognostic signature, then survival
cl = fcluster(linkage(Msig.values, "ward"), 2, "maxclust")
# log-rank: integrated class vs BAP1 mutation alone
p_class = logrank(surv(uvm_class))
p_bap1  = logrank(surv(bap1_status))
print(p_class, p_bap1)   # 2.9e-6   vs   0.54

95% of high-risk class 2 tumors have monosomy 3 with low BAP1 mRNA and protein, but only 30% carry a BAP1 mutation. Functional BAP1 loss is copy loss and silencing, which a mutation panel misses, so risk stratification should be multi-omic.
