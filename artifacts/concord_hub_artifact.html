Host the full site

The complete, cross-linked website

The artifacts above are standalone views. The full experience, a home search over 743 conditions plus the atlas, analyses, multi-modal, network, and every report, is a static site that hosts on GitHub Pages. It uses only relative links, so it runs at any path.

cd rarecancers && git remote add origin https://github.com/digvijayky/rare_cancer.git && git push -u origin master

Then enable Pages on the repo (branch master, root). Live at digvijayky.github.io/rare_cancer.

Anveshar is decision support for research and education, not medical advice. Confidence and validation scores summarize evidence strength, not the probability of benefit for any individual, and a driver-gene match is a translation hypothesis that variant context and tumor biology can overturn. Every diagnostic and treatment decision must be made by a qualified health care provider, ideally within a clinical trial.
